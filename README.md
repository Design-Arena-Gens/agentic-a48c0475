# Was kann ich damit machen?

Interaktive Ideenwerkstatt, die dir Inspiration liefert, was du mit deinem Projekt umsetzen kannst. Gebaut mit Next.js 14, React 18 und TypeScript.

## Lokale Entwicklung

```bash
npm install
npm run dev
```

Die App läuft standardmäßig auf `http://localhost:3000`.

## Produktion bauen

```bash
npm run build
npm start
```

## Projektstruktur

```
app/           # App Router Seiten & Layout
app/globals.css# Globale Styles
next.config.mjs# Next.js Konfiguration
```
